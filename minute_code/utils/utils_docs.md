# 유틸리티 모듈 (`utils/`)

`utils` 폴더는 프로젝트 전반에 걸쳐 공통적으로 사용되는 유틸리티 함수, 설정, API 키 관리 및 프롬프트 정의 등을 포함합니다. 특정 기능에 종속되지 않고 여러 모듈에서 재사용될 수 있는 코드들을 모아두어 코드의 재사용성과 관리 효율성을 높입니다.

## 주요 역할
- **API 키 관리**: `.env` 파일에서 필요한 API 키를 안전하게 로드하고 유효성을 검사합니다.
- **전역 설정**: 프로젝트의 디렉토리 이름, 사용 가능한 LLM 목록, 기본 회의 주제 및 키워드 등 전역적으로 사용되는 설정 값들을 정의합니다.
- **파일 입출력**: 처리된 결과를 파일 시스템에 저장하는 기능을 제공합니다.
- **프롬프트 정의**: LLM 및 STT에 사용되는 다양한 프롬프트 템플릿들을 중앙에서 관리합니다.

## 주요 파일
- `api_keys.py`: `PYANNOTE_TOKEN`, `OPENAI_API_KEY`, `GOOGLE_API_KEY`, `TAVILY_API_KEY` 등 외부 서비스 연동에 필요한 API 키를 로드하고 유효성을 확인하는 함수들을 제공합니다.
- `config.py`: `AUDIO_INPUT_DIR_NAME`, `RESULTS_DIR_NAME`, `AVAILABLE_LLMS` 등 프로젝트의 전반적인 설정 상수들을 정의합니다.
- `file_manager.py`: STT 결과, 교정된 텍스트, 요약 등 처리된 결과물들을 지정된 디렉토리 구조에 따라 파일로 저장하는 함수를 포함합니다.
- `prompts.py`: LLM(GPT, Gemini)의 텍스트 교정 및 요약, 그리고 STT에 사용되는 프롬프트 템플릿들을 변수 형태로 정의하여 관리합니다.
