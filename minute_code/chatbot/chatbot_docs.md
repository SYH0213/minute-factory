# 챗봇 모듈 (`chatbot/`)

`chatbot` 폴더는 저장된 회의록을 기반으로 질문에 답변하는 CRAG(Corrective RAG) 챗봇 기능을 구현합니다.

## 주요 역할
- **CRAG 파이프라인 구현**: 사용자의 질문에 대해 관련 문서를 검색하고, 필요시 웹 검색을 통해 정보를 보강하여 답변을 생성하는 복합적인 RAG(Retrieval Augmented Generation) 로직을 처리합니다.
- **벡터 저장소 연동**: `ChromaDB`와 연동하여 처리된 회의록 텍스트를 저장하고 검색합니다.
- **챗봇 프롬프트 관리**: 챗봇의 동작 방식과 답변 생성에 사용되는 프롬프트들을 정의하고 관리합니다.

## 주요 파일
- `crag_logic.py`: CRAG 파이프라인의 핵심 로직을 포함합니다. 문서 검색, 관련성 평가, 질문 재작성, 웹 검색, 답변 생성 등의 단계를 LangGraph를 사용하여 구현합니다.
- `chatbot_prompts.py`: 챗봇의 관련성 평가, 질문 재작성 등 CRAG 파이프라인에서 사용되는 시스템 및 사용자 프롬프트 템플릿들을 정의합니다.
