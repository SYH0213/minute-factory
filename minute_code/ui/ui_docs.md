# UI 모듈 (`ui/`)

`ui` 폴더는 Gradio를 기반으로 하는 웹 애플리케이션의 사용자 인터페이스(UI)와 관련된 모든 코드를 포함합니다.

## 주요 역할
- **UI 레이아웃 정의**: 애플리케이션의 전체적인 화면 구성과 각 컴포넌트(탭, 버튼, 텍스트 박스 등)의 배치를 담당합니다.
- **사용자 입력 처리**: 파일 업로드, 음성 녹음 등 사용자로부터 오디오 데이터를 입력받는 로직을 처리합니다.
- **이벤트 핸들링**: UI 컴포넌트에서 발생하는 이벤트(버튼 클릭, 값 변경 등)를 감지하고, 이에 따른 백엔드 로직(예: 처리 파이프라인 실행, 챗봇 응답 생성)을 호출합니다.

## 주요 파일
- `app_layout.py`: Gradio `gr.Blocks`를 사용하여 애플리케이션의 전체 UI 레이아웃을 정의하고, 각 UI 컴포넌트와 백엔드 콜백 함수를 연결합니다.
- `input_handlers.py`: 오디오 파일 업로드, 마이크 녹음 저장 등 오디오 입력과 관련된 구체적인 로직 및 Gradio 컴포넌트 업데이트 함수를 포함합니다.
