# Minute Factory: AI 회의록 자동 정리 프로젝트

이 프로젝트는 음성 회의록을 자동으로 텍스트로 변환하고, 화자를 분리하며, 내용을 교정하고 요약하는 AI 기반 웹 애플리케이션입니다.

여러 팀원(유수현, 성정현, 손용훈)이 각자 개발한 입력, 처리 모듈을 하나로 통합하고 개선한 버전입니다.

## 주요 기능

- **웹 기반 UI**: Gradio를 사용하여 누구나 쉽게 사용할 수 있는 웹 인터페이스를 제공합니다.
- **다양한 입력 방식 지원**:
    - **파일 업로드**: `mp3`, `m4a`, `mp4` 등 다양한 오디오/비디오 파일을 업로드하면 자동으로 `wav` 음성 파일로 변환됩니다.
    - **실시간 녹음**: 웹페이지에서 직접 마이크를 사용해 음성을 녹음하고 저장할 수 있습니다.
- **AI 처리 파이프라인**:
    - **화자 분리**: `pyannote.audio`를 사용해 대화에 참여한 여러 명의 화자를 식별합니다.
    - **음성 인식(STT)**: `OpenAI Whisper`를 사용해 음성을 텍스트로 정확하게 변환합니다.
    - **자동 교정 및 요약**: 처리된 텍스트를 LLM을 통해 문맥에 맞게 교정하고, 전체 회의 내용을 핵심만 요약합니다.
- **LLM 모델 선택**: 교정 및 요약 작업에 사용할 AI 모델을 `GPT-4`와 `Gemini-Pro` 중에서 선택할 수 있습니다.
- **결과 저장**: 원본, 교정본, 요약 등 모든 결과물을 체계적으로 폴더에 저장하여 관리합니다.

## 프로젝트 구조

```
minute_factory/
├── processing/           # 핵심 처리 로직 모음
│   ├── api_helpers.py    # API 키 로드 헬퍼
│   ├── audio_processing.py # 화자 분리 및 STT
│   ├── config.py         # 프로젝트 설정
│   ├── file_io.py        # 결과 파일 저장
│   ├── llm_handler.py    # GPT/Gemini 로직 처리
│   └── pipeline.py       # 전체 처리 과정 총괄
├── data/                 # (자동생성) 입력 오디오 파일 저장
├── results/              # (자동생성) 결과물 저장
├── temp/                 # (자동생성) 임시 파일 저장
├── .env                  # (직접생성) API 키 저장 파일
├── main_app.py           # UI를 포함한 메인 실행 파일
└── requirements.txt      # 필요 라이브러리 목록
```

## 설치 및 실행 방법

1.  **프로젝트 폴더로 이동**:
    ```bash
    cd E:\github\minute_factory
    ```

2.  **(권장) 가상 환경 생성 및 활성화**:
    ```bash
    python -m venv venv
    .\venv\Scripts\activate
    ```

3.  **필요 라이브러리 설치**:
    ```bash
    pip install -r requirements.txt
    ```
    *   **참고**: `torch`나 `ffmpeg` 설치에 문제가 발생하면 `requirements.txt` 파일 안의 주석을 참고하여 수동으로 설치해주세요.

4.  **API 키 설정**:
    *   `minute_factory` 폴더에 `.env` 라는 이름의 파일을 만드세요.
    *   파일을 열고 아래와 같이 필요한 API 키를 입력하세요.
        ```
        OPENAI_API_KEY="sk-..."
        GOOGLE_API_KEY="AIza..."
        PYANNOTE_TOKEN="hf_..."
        ```

5.  **애플리케이션 실행**:
    ```bash
    python main_app.py
    ```

6.  **웹 브라우저에서 접속**:
    *   터미널에 표시되는 주소(보통 `http://127.0.0.1:7860`)를 웹 브라우저에 열어 애플리케이션을 사용합니다.

## 사용 방법

1.  **1. 파일 업로드 및 관리** 탭 또는 **2. 음성 녹음** 탭을 사용하여 처리할 음성 파일을 `data` 폴더에 준비합니다.
2.  **4. 처리 & 요약** 탭으로 이동합니다.
3.  **처리할 오디오 파일** 드롭다운에서 원하는 파일을 선택합니다.
4.  회의 **주제**와 **주요 키워드**를 입력합니다. (선택 사항)
5.  사용할 **LLM 모델**(`gpt-4` 또는 `gemini-pro`)을 선택합니다.
6.  **처리 시작** 버튼을 클릭하고 작업이 완료될 때까지 기다립니다.
7.  작업이 완료되면 **처리 결과 보기** 섹션에서 요약 및 교정된 대화록을 확인할 수 있습니다. 결과 파일은 `results` 폴더에 저장됩니다.
